## 🔗 第四步：粘贴订阅链接 & 填写备注

1. 在输入URL框粘贴你的订阅链接  
2. 名称可随便写，例如「我的订阅」  
3. 点击 新增订阅源，确定即可
4. 如果显示节点列表代表导入成功

<img width="590" height="586" alt="image" src="https://github.com/user-attachments/assets/2781ea18-6239-48bb-8ecd-f502ecbfcea4" />

---

## 🟢 第五步：启动连接

回到首页：

1. 如果导入成功，那么请选中下方显示的随别一个节点 
2. 滑动底部开关，如果第一次使用，会出现图片里的弹窗
3. 显示 **绿色** 即成功
